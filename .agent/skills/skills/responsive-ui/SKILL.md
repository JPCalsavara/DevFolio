# Responsive UI Skill (MUI)

## Regras
1. Use a prop `sx` do Material-UI para definir responsividade usando breakpoints (`xs`, `sm`, `md`, `lg`, `xl`).
2. Prefira `Stack` e `Grid` (ou `Grid2`) do MUI para gerenciar layouts responsivos facilmente.
3. Teste garantindo que não há quebras no layout em telas de celular (xs).
