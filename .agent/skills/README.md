# Skills do Projeto

Este diretório contém todas as skills do projeto unificadas. As skills que antes ficavam em `.github/skills` foram migradas para cá, e adaptadas para o stack Next.js/React/Supabase.

## Skills Disponíveis

*   **`react-component`**: Regras para Next.js App Router utilizando Material-UI (MUI).
*   **`nextjs-api`**: Padrões para Next.js Server Actions/Route Handlers integrados com Supabase.
*   **`supabase-workflow`**: Gerenciamento de tabelas e seeds no diretório `/supabase`.
*   **`testing-workflow`**: Orientações de testes (usando Cypress/Playwright).
*   **`responsive-ui`**: Responsividade baseada no `sx` do Material-UI.
*   **`curriculo-latex-assistant`**: Assistente de extração de CV LaTeX.
*   **`preencher-staging-markdown`**: Preenchimento de dados locais em Markdown.
*   **`publicar-staging-supabase`**: Migração de staging para Supabase DB.
*   (Outras de workflow nativo: `plan`, `execute`, `bug-workflow`, etc)
