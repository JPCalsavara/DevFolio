# Content Staging

Use esta pasta para preparar conteudo antes de subir para o Supabase.

## Estrutura

- `markdown/`: textos em `.md` para projetos, experiencias e habilidades.
- `images/projects/`: imagens de projetos.
- `images/experiences/`: imagens de experiencias.
- `images/habilidades/`: icones/logos de habilidades.

## Fluxo sugerido

1. Monte os textos em `markdown/`.
2. Salve as imagens nas pastas corretas.
3. Faça upload pelo admin ou via dashboard do Supabase Storage.
4. Revise links/public URLs antes de salvar no banco.

## Modelos prontos

Use estes arquivos como base para o agente de IA preencher o portfolio:

- `markdown/project-template.md`
- `markdown/experience-template.md`
- `markdown/habilidade-template.md`
